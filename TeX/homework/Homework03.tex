\begin{homeworkProblem}
Suppose we are only given random drawings $\theta^{(0)}, \ldots, \theta^{(n)}$ from the Ga($\hat{a}, \hat{b}$) distribution. Then we can look at the 2.5\% quantile and 97.5\% quantile of these random drawings (in R we use the \texttt{quantile} command) in order to estimate $\theta_L$ and $\theta_U$.
\end{homeworkProblem}

\begin{homeworkProblem}
Now we suppose that $a = b = 1$, and $\sum y_i = 325, n = 250$, so our posterior distribution is Ga(326, 251). 
\begin{enumerate}[(a)]
\item
If we use the inverse CDF function (\texttt{qgamma} in R), then we can calculate the 95\% credible interval. In this case, our credible interval is (1.161627, 1.443528).
\item
If we are only allowed to use random samples from the gamma distribution (\texttt{rgamma} in R), then we simply find the quantiles of these samples. If we draw 250 random samples, we have a 95\% credible interval of (1.168850, 1.451595).
\item
Our MLE estimate is $\sum y_i / n = 1.3$. This is contained within both credible intervals we created. Also, the intervals from parts (a) and (b) are approximately equal.
\end{enumerate}
\end{homeworkProblem}
